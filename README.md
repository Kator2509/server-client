# Система общения между сервером и клиентом. С расширенным функционалом.

## В скором времени...

> Внимание, защита генерируется на уровне шума Перлина. При первом запуске программа
> выдаст вам уникальный ключ, которые требовательно сохранить и проставлять в конфигураторе
> для корректной работы SSL сервера. Вы сами несете ответственность за то, как храните ключ сервера.

> При попытке повторного вызова функции, метода или создания объекта сгенерирует <u><b>НОВЫЙ</b></u>
> уникальный ключ, который уже не будет подходить к сгенерированному сертификату и будет требоваться
> пересоздания ключа. В связи с этим вам придется сбрасывать <u><b>ВЕСЬ СЕРВЕР</b></u>.
